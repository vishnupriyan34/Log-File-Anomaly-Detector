
1. What is a Loop?

A loop repeats a block of code multiple times.

Types of Loops in C/Java
for
while
do while
2. for Loop
Syntax
for(initialization; condition; increment)
{
    // code
}

Example
for(int i=1;i<=5;i++)
{
    printf("%d",i);
}



3. while Loop
Syntax
while(condition)
{
   // code
}


Example
int i=1;

while(i<=5)
{
   printf("%d",i);
   i++;
}


4. do while Loop
Syntax
do
{
   // code
}while(condition);
Example
int i=1;

do{
   printf("%d",i);
   i++;
}while(i<=5);
Difference Between Loops
Loop	Condition Check	Executes Minimum
for	Beginning	0 times
while	Beginning	0 times
do while	End	1 time
5. Nested Loop

Loop inside another loop.

Example
for(int i=1;i<=3;i++)
{
   for(int j=1;j<=3;j++)
   {
      printf("* ");
   }
   printf("\n");
}



6. Infinite Loop

Condition never becomes false.

Example
while(1)
{
   printf("Hello");
}



7. break Statement

Stops loop immediately.

Example
for(int i=1;i<=5;i++)
{
   if(i==3)
      break;

   printf("%d",i);
}



8. continue Statement

Skips current iteration.

Example
for(int i=1;i<=5;i++)
{
   if(i==3)
      continue;

   printf("%d",i);
}





Big-O Notation

Common Time Complexities
Complexity	Meaning
O(1)	Constant
O(log n)	Logarithmic
O(n)	Linear
O(n log n)	Efficient sorting
O(n²)	Nested loop
O(2ⁿ)	Exponential
9. O(1) – Constant Time

Execution time never changes.

Example
int a=10;
printf("%d",a);



10. O(n) – Linear Time

Single loop.

Example
for(int i=0;i<n;i++)
{
   printf("%d",i);
}



11. O(n²) – Quadratic Time

Nested loops.

Example
for(int i=0;i<n;i++)
{
   for(int j=0;j<n;j++)
   {
      printf("*");
   }
}



12. O(log n) – Logarithmic

Problem size halves each step.

Binary Search Example
while(low<=high)
{
   mid=(low+high)/2;

   if(arr[mid]==key)
      break;
}


13. O(n log n)

Used in:

Merge Sort
Quick Sort

Efficient sorting algorithms.

14. Time Complexity Examples
Code	Complexity
Single statement	O(1)
One loop	O(n)
Two nested loops	O(n²)
Loop dividing by 2	O(log n)
